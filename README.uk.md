# GameTime Tracker

[English](README.md) | **[Українська](README.uk.md)** | [Русский](README.ru.md)

Хомбрю для Nintendo Switch, що показує скільки часу ви провели у своїх іграх.

Протестовано на прошивці **20.5.0** — працює стабільно.

> **Наразі вихідний код репозиторію закритий.** Я допрацьовую й причепурюю проєкт, і відкрию доступ до src, коли він буде готовий. Скомпільований `.nro` файл для встановлення доступний у розділі Releases.

## ⚠️ Дисклеймер

Потрібна прошита або чіпована консоль.

## Що вміє программа

1. Після запуску программа завантажує дані про зіграний час у всіх встановлених іграх, а тоді відкриває головне меню зі списком ігор і часом, проведеним у кожній.
2. Зліва від назви гри — загальна кількість днів, справа — час у годинах.
3. Вибравши гру і натиснувши **A**, ви потрапляєте у меню гри, де можна погортати дні та подивитись, скільки часу було проведено в грі в кожен конкретний день.

## Керування

**Головне меню:**
- Хрестовина (вгору/вниз) або лівий стік — навігація по списку ігор
- **A** — відкрити деталі гри
- **B** — вийти з программи

**Меню гри:**
- Хрестовина або лівий стік — гортати дні
- **B** — повернутись у головне меню

## Плани на майбутнє

Программа розвиватиметься поступово:

- [ ] Перегляд часу, проведеного у вже видалених іграх
- [ ] Можливість змінювати шрифт
- [ ] Підтримка інших мов
- [ ] Виправлення багів та дрібні покращення за потреби

## Встановлення

1. Відкрийте **DBI**, оберіть **"Запустити MTP зв'язок"** — на комп'ютері відкриється папка консолі.
2. Перейдіть за шляхом `Switch\1: SD Card\switch`. Рекомендується створити тут окрему папку `switch-playtime` і закинути туди `.nro` файл із розділу Releases.
3. Дочекайтесь завершення передачі файлу та вийдіть із DBI.
4. Якщо у вас трохи застаріла прошивка (як у мене), заходьте в Homebrew Menu (наприклад, **Sphaira**), обирайте **GameTime Tracker**, натискайте **X** для меню опцій і вибирайте **"Встановити форвардер"** — іконка з'явиться прямо на робочому столі Switch.
